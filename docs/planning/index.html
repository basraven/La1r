          What’s next?

          Since I’m (currently) only developing la1r by myself, there are only so many things you can do at once. (Feel free to reach out through Github if you want to get involved!) For this reason I created this planning page in which I track and prioritize what I will add to la1r next. Feel free to add comments on this through Github!
